## 공급자별 응답 계약 확인

### Kakao Local

- 키워드 검색과 카테고리 검색 모두 장소명, 주소, 좌표, 카테고리, 장소 링크를 안정적으로 반환했습니다.
- `road_address_name`이 비어 있는 경우가 있어 `address_name` 대체가 계속 필요합니다.
- 장소 좌표는 문자열이므로 숫자 변환과 대한민국 범위 검증이 필요합니다.

### ODsay

- 대중교통 도달권은 `Polygon`/`MultiPolygon`으로 정규화해 교집합 계산에 사용할 수 있었습니다.
- 공통 영역이 없는 경우는 오류가 아니라 정상 결과 `NO_INTERSECTION`으로 반환됐습니다.

### TMAP Transit

- 거점 평가와 공동 후보 재평가 모두 정상적으로 이동시간/환승 정보를 반환했습니다.
- HTTP 200만으로는 충분하지 않으므로 본문 status 판정은 계속 유지해야 합니다.
